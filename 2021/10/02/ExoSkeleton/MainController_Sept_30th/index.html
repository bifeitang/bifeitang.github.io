# Test Setup

  • Main Controller

    • Send two types of command

      • Heart Beat Command

        • CMD ID: 0x02x to 0x07x
        • CMD Argument: 0x31x, 0x00, 0x00, ... 0x00 (8 bytes)

      • Joint Control Command

        • For each joint, main controller will send angle, velocity, torque three commands each time
        • Command argument Angle , Velocit , Torque are just place holders, the actual value should be float (32 bits).

# Problems

# The main controller state machine will sometime stops after running for a few minutes.

  • Reason

    • Debug Print function is shared by all tasks, the race condition during lock acquiring with naïve implementation can cause unbounded waiting.
    • The communication between State Machine Task and Command management tasks are through FreeRTOS queue, there are cases when the print task (basing on interrupt) happens during enqueue/dequeue that blocks the queue.

  • Solution

    • Use FreeRTOS lock for the print function
